What is Github ❓

Github is tool which uses git. Github gives us all functionalities of git, enables us share code with others and collaborate with other programmers.

Some useful commands

  • git init: used to initialize empty git repository
  • git clone <path> : this creates a working copy of a git repository from a remote source to your local repository.
  • git commit -m <msg> : used to commit changes to your local git repository
  • git push: used to push changes to your local repository
  • git checkout -b branch_name: it create a branch if it does not exist and switch to that branch.
  • git status: used to get current state of your branch
  • git branch: check currnet branch you'r eworking on
